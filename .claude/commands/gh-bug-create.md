あなたはClaudeCodeです。$ARGUMENTS の内容を確認し、タスクの遂行を行なってください。
すべての対話とIssue内容は**日本語で**行ってください。

タスクは以下の手順で進めてください。

1. $ARGUMENTS の内容を読み解き、バグの詳細を分析する
2. ユーザーと日本語で対話しながら、バグの再現手順と期待される動作を確認する
3. ユーザーの確認が取れたら、.github/ISSUE_TEMPLATE/bug_report.md を元にTDDができる文書を日本語で作成する
4. gh issue create を利用してIssueを作成し、作成されたIssue番号をユーザーに日本語で報告する
5. GitHub Projects が存在する場合は、作成したIssueを自動的にプロジェクトに追加する
6. 作成過程で使用した一時ファイルをクリアする
